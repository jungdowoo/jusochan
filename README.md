# 🏠 Jusochan: 도로명 주소 다국어 번역 서비스



Jusochan은 외국인을 위한 **도로명 주소 다국어 번역 웹사이트**입니다.  
사용자가 입력한 한국 도로명 주소를 **영어 / 일본어 / 중국어**로 자동 번역해주며,  
📦 국제 택배, 🏠 이사, 📨 외국인 등록증, 💸 해외 송금 등 다양한 상황에서 활용할 수 있습니다.

---

## 🔗 사이트 주소

👉 [jusochan.com](http://jusochan.com)

---

## 💡 주요 기능

- ✅ 한국 도로명 주소 → 영어 / 일본어 / 중국어 실시간 번역
- ✅ 공공데이터 API + Google Translate API 연동
- ✅ SEO 최적화 (Google 검색 노출 고려)
- ✅ AWS EC2 배포 및 도메인 연결
